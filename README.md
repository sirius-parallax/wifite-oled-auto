# 📡 WiFi Audit Tool with OLED Display

Автоматизированный инструмент для аудита безопасности WiFi сетей на базе одноплатного компьютера **NanoPi NEO** с выводом информации на **OLED дисплей**. Предназначен для тестирования собственных сетей на устойчивость к атакам.

## 📋 Описание

Скрипт автоматически проводит трёхэтапный тест безопасности WiFi сетей:
1.  **WPS Pixie-Dust** — проверка на уязвимость WPS.
2.  **WPA Dictionary + PMKID** — атака по словарю с использованием PMKID.
3.  **WPA Dictionary (Handshake Only)** — классическая атака по словарю на основе захвата рукопожатия.

Вся информация о процессе атаки, найденных сетях и результатах выводится в реальном времени на OLED дисплей 128x64, что позволяет использовать устройство автономно без подключения монитора.

---

## 🖥️ Демонстрация работы OLED

Дисплей автоматически переключает режимы отображения каждые 5 секунд:

| Режим | Описание | Пример |
|-------|----------|--------|
| **📊 Статус атаки** | Текущая атака, таймеры, цель | `WPS Pixie-Dust / Cur: 02m15s` |
| **📜 Логи** | Последние 4 строки вывода wifite | `[+] Scanning. Found 8 targets` |
| **🔓 Взлом** | Мигает при нахождении пароля | `!!! CRACKED !!! / SSID: ... / KEY: ...` |
| **⚙️ Результаты** | Итоги после завершения атаки | `Found: 1 networks` |

## 🛠️ Используемое оборудование

| Компонент | Модель / Характеристика |
|-----------|-------------------------|
| **Одноплатный ПК** | NanoPi NEO (или аналог на Allwinner H3/H5) |
| **WiFi адаптер** | Совместимый с режимом монитора (RTL8812AU, AR9271 и др.) |
| **Дисплей** | OLED SSD1306 128x64 (I2C) |
| **Питание** | 5V 2A (MicroUSB или GPIO) |
| **ОС** | Armbian Linux (Debian/Ubuntu) |
